# Portfolio Jr - exemplo simples

Este projeto foi feito para ser facil de entender e editar.

## Estrutura

- `index.html`: estrutura da pagina (textos, secoes, links)
- `styles.css`: visual (cores, espacamento, responsividade)
- `script.js`: comportamento simples (menu mobile, lista de projetos, copiar e-mail)

## Como abrir

1. Clique duas vezes no arquivo `index.html`.
2. Ou use uma extensao de servidor local no VS Code (ex.: Live Server).

## O que editar primeiro

1. Nome, cargo e descricao na secao inicial (`index.html`).
2. E-mail, LinkedIn e cidade na secao de contato (`index.html`).
3. Lista de projetos no array `projects` (`script.js`).
4. Cores em `:root` (`styles.css`) para personalizar o tema.

## Dica

Sempre altere uma parte por vez e recarregue a pagina para validar.
Assim voce aprende mais rapido e evita quebrar tudo de uma vez.
